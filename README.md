**Finding the hidden potential in your room.**  

通过智能化的视觉洞察，唤醒空间的隐藏潜力，让整理变得简单、愉悦且富有美感。

## ✦ 产品理念 | Philosophy

DeclutterAI 不仅仅是一个工具，它是生活方式的催化剂。我们通过重塑物理空间，帮助用户找回生活的主导权。

- **化繁为简 (Actionable Simplicity)**  
  将庞大的整理工程拆解为“断舍离、收纳、陈设”三步走，消除启动焦虑。

- **发现潜力 (Potential-Focused)**  
  非评判性（Non-judgmental）的引导，不批评现状，只描绘未来。

- **空间美学 (Functional Aesthetics)**  
  结合色彩调色板（Color Palette）建议，让整理上升为室内设计。

- **专业平民化 (Democratizing Design)**  
  搭载 Gemini 3 Flash，让每个人都拥有私人空间顾问。

---

## ✦ 核心体验 | Core Experience

### 1️⃣ 优先级任务系统

不再面对杂乱无章。AI 自动识别并划分：

- 🔴 **High**: 立刻处理（如：过期的外卖盒）  
- 🟡 **Medium**: 需要归位（如：散落的书籍）  
- 🟢 **Low**: 审美升级（如：调整装饰画位置）

---

### 2️⃣ 视觉美学引擎

利用多模态大模型提取空间色彩，提供《Kinfolk》风格的配色方案建议。

---

### 3️⃣ 极简 UI 设计

- **字体方案**: Playfair Display（标题） + Inter（正文）  
- **视觉重心**: 大面积留白，杂志化排版  

---

## ✦ 技术实现 | Technical Stack

- **Model**: Google Gemini 3 Flash（Multimodal）  
- **Framework**: Vibe Coding with AI Studio  
- **Design Language**: Minimalist / Magazine Style
